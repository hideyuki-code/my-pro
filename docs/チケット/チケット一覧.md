# ポケモン情報閲覧アプリ 開発チケット一覧（学習用・簡易版）

## 概要

このチケット一覧は、PokeAPIを利用したポケモン情報閲覧アプリの開発を進めるためのものです。
各チケットは学習を目的としており、約4時間で完了できる粒度に分割されています。
全体の開発時間は約100時間を想定しています。

## 学習要素

各チケットには以下の学習要素が含まれています：

### TypeScript関連
- 基本的な型定義（TypeScript 5.3.3基準）
- インターフェースの活用
- 型安全なイベントハンドリング

### React関連
- 関数コンポーネントの作成（React 18.2.0基準）
- Hooksの基本（useState, useEffect, useContext）
- カスタムフックの作成
- Context APIを使った状態管理

### API・非同期処理関連
- Axios（v1.6.8）によるAPI通信
- async/awaitによる非同期処理
- エラーハンドリング・ローディング管理
- LocalStorageの利用

### テスト関連
- Jest の基本
- React Testing Library の基本
- コンポーネントテストの書き方

### その他
- アクセシビリティの基本
- レスポンシブデザインの基本
